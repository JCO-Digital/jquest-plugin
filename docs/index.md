# JQUEST Plugin docs

The plugin makes it possible to add an organization id and fetch JQUEST games that can be inserted in site using a Gutenberg block.


## Installation ##

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings -> JQUEST Settings and add your organization id
4. Go to a page or post and add a JQUEST inserter block
